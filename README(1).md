Invoice Generator React App
==========================

Quick start:
1. unzip and cd into the project folder
2. run `npm install`
3. run `npm start`

This app uses:
- React (Create React App)
- html2canvas + jsPDF to export invoice to PDF

Letterheads are provided as placeholder PNGs in /public/letterheads.
Replace them with your real letterhead images (same filenames) if you have them.

Invoice naming, numbering, stamp, bank-selection, and multi-item support are implemented.
